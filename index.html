Command Reference Guide

The Analytical Engine processes a simple set of commands to perform calculations. Each command is executed sequentially, similar to how punch cards were fed into early computing machines.

Command Syntax Description
LOAD LOAD [var] [value] Initializes a variable with a specific value
ADD ADD [var] [value/var] Adds a value or another variable to the specified variable
SUB SUB [var] [value/var] Subtracts a value or another variable from the specified variable
MULT MULT [var] [value/var] Multiplies the specified variable by a value or another variable
DIV DIV [var] [value/var] Divides the specified variable by a value or another variable
PRINT PRINT [var] Outputs the current value of the specified variable
JUMP JUMP [line-number] Jumps to a specific line in the program
JUMPIF JUMPIF [var] [condition] [value] [line] Conditional jump if the comparison evaluates to true
STORE STORE [var] [address] Stores the variable value at a specific memory address
RECALL RECALL [var] [address] Retrieves a value from a memory address into a variable
// Calculate factorial of 5 LOAD N 5 LOAD RESULT 1 LOAD I 1 // Loop start LABEL LOOP MULT RESULT I ADD I 1 JUMPIF I <= N LOOP PRINT RESULT // Should output 120

Tips for Effective Programming

  • Use comments (lines starting with //) to document your program
  • Variable names are case-sensitive (A is different from a)
  • The engine executes one command at a time, from top to bottom
  • Use the Step-by-Step execution to understand complex program flow
  • Check the Memory Storage tab to observe how variables change
